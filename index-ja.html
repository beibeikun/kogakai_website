真正性証明・流通

データ、SaaS、標準化されたシステムを通じて、
高級品・美術品・骨董品を
「信頼可能・追跡可能・流通可能・金融化可能」な資産へ

核心問題と解決策

長年にわたり、世界の高付加価値資産は流通の過程において、真贋基準の不統一、データの高度な分散、取引の不透明性といった課題を抱えており、その結果、金融システムによる有効な識別および接続が困難な状況にありました。
こうした構造的なボトルネックに対し、当社はプラットフォーム型のアプローチにより、「真正性の検証」から「データの権利確定」、「データの蓄積」から「資産流通」、さらに「金融活用」へと至る一体型の基盤インフラを体系的に構築し、高付加価値資産がデジタル化・標準化・金融化の体系へと進入するための中核的な通路を切り拓いています。

五大核心能力目標

高価値資産が「信頼できる検証」から「金融活用」に至るまでのライフサイクル全体を見据え、当社は五つの中核能力の体系的な構築を進めています。 AI×XRF を中核とする科学的検証能力により資産の真正性を確立し、Micro-ID によって資産に唯一無二のデジタルアイデンティティを付与します。 さらに、国境を越えて連携可能な高価値資産データベースを段階的に蓄積し、業界向けに B2B SaaS および API レベルの基盤サービスを提供します。 その上で、資産の金融化インターフェースを探求し、金融システムから認識・評価・活用され得る基盤インフラの構築を目指します。

科学的検証能力
AI × XRF｜構築中

AIによる画像認識、素材特性分析、XRF非破壊検査技術を導入し、標準化・再現性のある科学的検証体系の構築を進めています。 本能力は、単一の専門家の経験への依存を低減し、「鑑定」を主観的判断から、定量化・追跡可能なデータ結果へと高度化することを目的とし、 高価値資産の真正性に対する技術的裏付けを提供します。

資産の唯一デジタルID
Micro-ID｜企画段階

検証を通過したすべての資産に対し、複製不可能な Micro-ID を付与し、 デジタル空間における唯一のアイデンティティとして管理する構想を進めています。 このIDは、検証結果、流通履歴、主要特性データと紐づけられ、 将来的な権利確定、追跡、流通および金融活用に向けた統一的な入口となります。

国際資産データベース
段階的に構築

法令遵守を前提として、地域やオークション形態を横断した高価値資産データを段階的に蓄積し、 構造化・検索可能・分析可能な国際資産データベースの構築を進めます。 本データベースは、業界内の業務効率向上にとどまらず、 資産評価、リスク管理、研究分野における重要なデータ基盤となることを目指しています。

B2B SaaS・API
開発中

オークション会社、鑑定機関、保管・流通プラットフォーム向けに、 B2B SaaS システムおよび標準化された API を提供します。 これにより、検証、データ管理、資産IDサービスへの低コストな接続を可能にし、 当社プラットフォームを単一の業務システムではなく、 業界全体を支える基盤インフラとすることを目指しています。

資産金融化インターフェース
研究段階

前述の各能力が成熟した段階を前提に、資産と金融システムを接続する 標準化インターフェースの研究を進めています。 権利証明、データ連携、リスク管理支援などを通じ、 高価値資産が融資、担保、証券化といった金融領域へ展開する可能性を創出しますが、 あくまで慎重かつ段階的な推進を基本方針としています。

理念驱动

古河會は「古の匠の精神を継承し、テクノロジーによって信頼を構築する」という使命のもと、 科学的検査とビッグデータ分析を融合し、統一的・客観的・検証可能な年代判定および来歴追跡の体系を構築しています。 元素分析、AI認識、ブロックチェーンによる権利確定などの先端技術を原動力として、 世界のアンティーク取引に対し、権威性と透明性を備えた価値基準を提供します。 また、国際的な相互接続を志向し、世界各地のオークションハウス、美術館、コレクション機関と相互承認可能な デジタル標準を確立することで、文化資産のグローバルな流通と価値転換能力の向上を目指しています。

私たちの使命

協働と共創を原則とし、オークションハウス、コレクター、保険機関、文化機関に対して、 鑑定から認証に至るまで、全工程・全ライフサイクルをカバーするサービスを提供します。

私たちのビジョン

流通するすべてのアンティークに「デジタルID」を付与し、 信頼によってコレクションが広がり、テクノロジーによって文化が継承される世界を実現します。

古河會について

古河會は、企業および機関投資家向けに、 高価値資産のデータおよび流通を担うプラットフォームの構築を準備しています。 本プラットフォームは、単一の鑑定サービスやオークション事業ではなく、 長期的な視点に立った基盤インフラ型ビジネスとして位置づけています。

株式会社古河會
